# Credit Risk Analysis & Prediction

This project focuses on analyzing credit data to understand the factors influencing loan defaults and building a predictive model to assess credit risk.

## Dataset Overview
The dataset (`1.csv`) includes demographic and financial attributes such as:
* **Person Attributes:** Age, Annual Income, Home Ownership, and Employment Length.
* **Loan Attributes:** Intent, Grade, Amount, and Interest Rate.
* **Target Variable:** Loan Status (Default vs. Non-Default).

##  Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
* **Environment:** Jupyter Notebook

##  Analysis Workflow
1. **Data Preprocessing:** Handling missing values and encoding categorical variables.
2. **Exploratory Data Analysis (EDA):** Visualizing correlations between income levels, loan amounts, and default rates.
3. **Modeling:** Implementing classification algorithms to predict the likelihood of loan approval or default.

##  How to Use
1. Clone this repository.
2. Ensure you have the required libraries installed via `pip install pandas matplotlib seaborn scikit-learn`.
3. Open `1.ipynb` in Jupyter Notebook or Google Colab to view the analysis.

##  Visualizations
The repository includes key plots (`.png` files) that highlight the relationship between interest rates and loan grades, providing insights into the risk profiles of borrowers.
